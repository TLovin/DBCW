Analyse viewer reaction to a film in relation to their rating history

Please enter the UserId of the user who's rating history you want to analyse and the Movie Name for which you want to analyse that user's reaction.

e.g. User ID: 1 and Movie Name: toy story will analyse user with UserID 1's reaction for toy story compared to their rating history.

Analyse viewer ratings by rating history and genres

Please enter the UserId of the User who's rating history you want to analyse, the genre for which you want to analyse the user's rating history and the Movie Name for which you want to analyse that user's reaction.

e.g. User ID: 1, Genre: fantasy and Movie Name: toy story will analyse user with UserID 1's reaction for toy story compared to their rating history in the fantasy genre.
