Search for films by Title+Date or Genre

If you want to search by Title+Date, then type the title of the movie followed by a "," and then the year of the movie and then select "Title+Year" from the drop down.

e.g. toy story, 1995

If you want to search by Year, then type in the year to search for and then select "Year" from the drop down.

e.g 1995

If you want to search by Genre, then type in the genre to search for and then select "Genre" from the drop down.

e.g action

If you want to search for something through all the fields, type it in and then select "All the fields" from the drop down.

Find Most/Least Popular Movies

Please type in the Genre, Year and whether you want to see most/least popular movies from the drop down.

e.g. Genre: action and Year: 1995
