Welcome to our SQL Database to do with Movies

Searching for Title+Date or Genre:

If you want to search by Title+Date, then type the title of the movie then a "," and then the year of the movie and then select "Title+Year" from the drop down.

If you want to search by Year, then type in the year to search for and then select "Year" from the drop down.

If you want to search by Genre, then type in the genre to search for and then select "Genre" from the drop down.



Finding Most/Least Popular Movie:

Please type in the Genre, Year and whether you want to see most/least popular movies from the drop down.
